# Flowdo – Work Organizer

To-Do-Listen, Kalender, Pomodoro-Timer und Boards/Kampagnen in einer Web-App (React + Vite + TypeScript + Tailwind).

## Voraussetzungen

Node.js (Version 18+) und npm müssen installiert sein: https://nodejs.org/

## Setup

```bash
npm install
npm run dev
```

Die App läuft danach unter http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Daten

Alle Daten (Aufgaben, Boards, Pomodoro-Sessions, Einstellungen) werden lokal im Browser (LocalStorage) gespeichert.
